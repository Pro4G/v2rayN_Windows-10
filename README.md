# Инструкция настройки v2rayN для Windows 10 и подключение подписок

Скачать программу в архиве по ссылке:

https://github.com/2dust/v2rayN/releases

Выбрать самый свежий релиз, зайти в него, прокрутить вниз и выбрать файл с названием v2rayN-windows-64-desktop.zip
<img width="1158" height="736" alt="image" src="https://github.com/user-attachments/assets/385819d5-728c-4be5-8ad3-503827304108" />
<img width="1235" height="860" alt="image" src="https://github.com/user-attachments/assets/b78913b0-1f5b-4c1c-bf6a-07df9e7128c9" />

Скачанный архив разпаковать в любой папке. Должен разархивироваться примерно такой список файлов:
<img width="1111" height="431" alt="image" src="https://github.com/user-attachments/assets/e8564d62-7be2-41dc-8df5-439f843eb937" />

В проводнике в каталоге с программой на файле v2rayN (приложение) нажать правую кнопку мыши и выбрать пункт Свойства. В открышемся окне поставить галочку напротив "Разблокировать" и нажать Применить. Перейти на вкладку Совместимость и поставить галочку "Запускать эту программу от имени администратора" и нажать ОК.
